== Installation ==

### Installation

1. Upload 'fb-post' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. You will find new button in WordPress post/page editor.

### Shortcode

* Just use this shortcode to embedding facebook post: [fb_pe url="" bottom=""]
* url="here paste facebook post link" required attribute.
* bottom="here enter value for margin bottom, numbers only, default is 30" not required attribute.

### example

	[fb_pe url="https://www.facebook.com/FacebookDevelopers/posts/10152128760693553"]
